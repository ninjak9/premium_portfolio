# Design System: Ethereal Noir

This document serves as the official source of truth for the **Portfolio Website** design system.

## 🎨 Color Palette
The design utilizes a "Deep-Space" palette centered on Atmospheric Precision and Glassmorphism.

| Token | Value | Description |
| :--- | :--- | :--- |
| **Background** | `#131313` | Primary surface background. |
| **Primary** | `#FFFFFF` | High-contrast typography and essential icons. |
| **Secondary** | `#A1A1A6` | Secondary labels and metadata. |
| **Tertiary/Accent** | `#0A84FF` | Interactive highlights and small indicators. |
| **Neutral** | `#000000` | Foundation for infinite depth and contrast. |
| **Surface Tint** | `#C6C6C7` | Used for subtle depth transitions. |

## 📐 Layout & Spacing
A rigid mathematical alignment based on an 8px base unit.

| Token | Value | Description |
| :--- | :--- | :--- |
| **Base Unit** | `8px` | The foundation for all spacing. |
| **Container Max** | `1280px` | Maximum width for content sections. |
| **Gutter** | `24px` | Space between grid columns. |
| **Margin** | `48px` | Minimum external margins. |
| **Card Gap** | `20px` | Gap between elements in Bento grids. |

## 💎 Visual Language
- **Shapes**: Nested squircles. Standard components use **16px** radius; large containers use **32px** radius.
- **Glassmorphism**: Surfaces use semi-transparent dark grays with **20px-40px** backdrop blur.
- **Borders**: 1px inner stroke with low-opacity white (8%-20%) to simulate glass edges.
- **Depth**: Established through layering and blurs rather than traditional shadows.
